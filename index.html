Desenvolvimento para Internet

  • O HTML (Hyper Text Markup Language) se trata de uma linguagem de larcação de hipertexto que tem como principal utilidade a criação de páginas na web. Essa é a linguagem que o seu navegador (Browser) decodifica para exibir as páginas.
  • Hipertexto pode ser definido como todo o conteúdo o qual se encontra inserido em um documento para a web, sendo que sua principal característica seria a possibilidade de se interligar a outros documentos também na web.
  • O HTML se baseia no conceito de hipertexto. Hipertexto são conjuntos de elementos – ou nós – ligados por conexões. Estes elementos podem ser palavras, imagens, vídeos, áudio, documentos etc.
  • Com a linguagem HTML, é possível desenvolver toda a estruturação do conteúdo de uma página (título, subtítulo, posicionamento de imagens, tabelas, etc).
  • As páginas criadas em HTML são visualizadas pelos navegadores.
