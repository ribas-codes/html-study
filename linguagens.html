CSS

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) uma linguagem de estilo (en-US) usada para descrever a apresentao de um documento escrito em HTML ou em XML (incluindo vrias linguagens em XML como SVG, MathML ou XHTML). O CSS descreve como elementos so mostrados na tela, no papel, na fala ou em outras mdias.
